## 📌 项目简介

**广州中考助手**是一个为2026届广州中考生和家长设计的免费志愿参考工具。通过5个简单问题，系统会根据你的分数、区域、住宿需求等条件，智能推荐匹配的高中学校。同时提供160+所广州高中近三年录取分数线、高考成绩、宿位信息、官方渠道等详细资料。

### ✨ 核心功能

- 🎯 **5问智能匹配**：回答分数、区域、公办/民办、住宿、学校类型，生成专属推荐列表
- 📚 **学校资料库**：160+所学校，包含省市属、区属龙头、民办等全梯队
- 📈 **高考成绩**：特控率、600分以上占比、清北录取人数等关键数据
- 🏷️ **标签筛选**：按梯队、区域、性质快速筛选学校
- 🔗 **官方直达**：各校官网、公众号一键跳转
- 📱 **移动优先**：完美适配手机端，家长群分享体验极佳
